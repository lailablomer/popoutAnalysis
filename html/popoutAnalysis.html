mousePosition

Takes a video file and returns the position of the mouse for every frame. also it scores the time (in number of frames) the mouse spents at each of the 10 stimuli.

function [positions, scoring, popout, rest, files] = popoutAnalysis(filename, startframe, endframe, draw, findCircle, includedArea)

filename = file to be analysed startframe, endframe = both integers draw = true or false: specifies whether you want to draw the ROI's yourself. excludedArea = integer, specifies the part of a circle excluded from analysis. If 0, the whole outer circle is analysed.
